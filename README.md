# Road-Accident-Dashboard
Shows the detailed analysis of road accident in past two years.
Project Overview

This project presents a Road Accident Analysis Dashboard developed in Microsoft Excel to analyze and visualize accident statistics.
The goal is to identify trends, understand contributing factors, and support data-driven decision-making for improving road safety and resource allocation.

Dashboard Summary

Total Casualties: 417,883

Fatal Casualties: 7,135 (1.7 %)

Serious Casualties: 59,312 (14.2 %)

Slight Casualties: 351,436 (84.1 %)

Casualties by Car: 333,485 (79.8 %)

Key Insights

Vehicle Type: Cars account for the majority of casualties, followed by motorcycles and goods vehicles.

Road Type: Single carriageways represent the highest number of accidents (~309 K).

Location & Lighting: Urban and daylight conditions show higher incident counts, indicating traffic density as a major factor.

Monthly Trends: Comparative analysis of current-year vs. previous-year casualties highlights seasonal and behavioral variations.
